Extract the data

Extracted the data from CDC: Centers for Disease Control and Prevention:

Deaths involving coronavirus disease 2019 (COVID-19), pneumonia, and influenza reported to NCHS by week ending date and by state

Link to datasource: https://data.cdc.gov/NCHS/Provisional-COVID-19-Death-Counts-by-Week-Ending-D/r8kw-7aab.

Data was available for export in various different formats CSV, JSON, RDF, RSS, XML etc. For the purpose of this project Json data was exported.
